# Weather Alert System 🌦️

## Description
This project is a Python-based Weather Alert System that analyzes environmental data and provides alerts for potential disaster risks such as heatwaves, floods, and extreme weather conditions.

## Features
- User input for city, temperature, rainfall, humidity
- Alert generation system
- Data storage using pandas
- Data visualization using matplotlib

## Technologies Used
- Python
- Pandas
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python main.py

## Purpose
This project demonstrates the use of data analysis and visualization in disaster preparedness and early warning systems.
